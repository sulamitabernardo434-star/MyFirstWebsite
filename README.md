
# Brownie Recipe Website (HTML & CSS)

## Overview

This project is a responsive recipe website created using HTML and CSS as part of my Introduction to Software Technology coursework. It demonstrates the fundamentals of front-end web development by presenting a visually styled webpage with images, formatted text, lists, hyperlinks, and custom CSS styling.

The website provides a brownie recipe with ingredients, step-by-step instructions, and a link to a video tutorial.

## Features

* Structured webpage using HTML
* Custom styling with CSS
* Recipe ingredients and instructions
* Images throughout the webpage
* External hyperlinks
* Organized page layout with headings and lists

## Technologies Used

* HTML5
* CSS3

## Concepts Demonstrated

* HTML page structure
* CSS styling
* Typography and color customization
* Images and hyperlinks
* Ordered and unordered lists
* Div containers and page layout
* Basic web design principles

## How to View

1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.

No additional software or installation is required.

## Learning Outcomes

Through this project, I developed foundational front-end web development skills by learning how to structure webpages with HTML and enhance their appearance using CSS. This project strengthened my understanding of webpage organization, styling, and user-friendly design.
